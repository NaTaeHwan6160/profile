프로그램 명: 모바일 통합검색 키워드 점유 체크 프로그램
제작자 : 나태환
제작일 : 241227
프로그램 동작 설명:
input.txt를 업데이트할때 사용하는 시트

our_url.txt를 업데이트 할 때 사용하는 시트

해당 스프레드 시트에 적혀있는 게시글 주소를 읽고 모바일 통합검색 결과에 해당 게시글이 있는지 확인

있으면 점유 없으면 탈환필로 결과물에 적힘

결과물은 output폴더에 해당 날짜 명으로 html 파일이 생성됨

[프로그램 실행에 필요한 파일]
input.txt - 검색 키워드 내용이 적혀있는 텍스트 파일
our_url.txt - 우리 게시글 주소가 적혀있는 텍스트 파일

[실행파일 설명]
0_run.bat -> input.txt, our_url.txt를 읽고 실행
1_run.bat -> our_url.txt의 내용을 스프레드시트를 읽고 업데이트 한 뒤 실행됨
2_run.bat -> input.txt의 내용을 스프레드시트를 읽고 업데이트, our_url.txt의 내용을 스프레드시트를 읽고 업데이트 한 뒤 실행됨
3_run.bat -> input.txt의 내용을 스프레드시트를 읽고 업데이트한 뒤 실행됨
